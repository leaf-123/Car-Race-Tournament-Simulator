# Racing Tournament Simulator

A Java object-oriented programming project that simulates a racing tournament across multiple race tracks.

## Features

- Driver, vehicle, mechanic, and race track system
- Vehicle lifecycle states:
  - OPTIMAL
  - WORN
  - CRITICAL
  - RETIRED
- Weather effects on vehicle performance
- Pit stop repair and refueling logic
- Race result calculation and tournament scoring
- Final podium ranking system
- JUnit tests for core logic and edge cases

## Technologies

- Java
- JUnit 4
- Object-Oriented Design

## Project Structure

```text
src/racing/       -> main implementation
test/racing/      -> JUnit tests
```

## Example Simulation Flow

1. Load tournament data
2. Create race events for tracks
3. Assign drivers, vehicles, and mechanics
4. Simulate races lap by lap
5. Handle vehicle condition changes and pit stops
6. Calculate race results and points
7. Display final podium

## Notes

This project focuses on:
- object-oriented design
- class relationships
- state-based logic
- modular structure
- unit testing
